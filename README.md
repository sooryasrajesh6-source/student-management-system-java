# Student Management System (Java)

## Overview

This is a simple Student Management System developed in Java using Object-Oriented Programming (OOP) concepts. The application runs in the console and allows users to manage student records efficiently.

## Features

* Add Student
* View All Students
* Search Student by ID
* Menu-Driven Interface
* Object-Oriented Design
* User Input Handling with Scanner

## Technologies Used

* Java
* OOP Concepts

  * Classes and Objects
  * Constructors
  * Encapsulation
  * Arrays
* Scanner Class

## Project Structure

* `Student` class for storing student details
* `StudentManager` class for managing student records
* `Main` class for user interaction
